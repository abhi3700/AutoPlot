# AutoPlot GUI
## Features
* AutoPlot has a GUI for user to interact.
* All the data is moved from Excel to Cloud (local database).
* The brightness level is customized to 3 themes - Light, Dark, Cleanroom

## Themes
* Light
* Dark
* Cleanroom
